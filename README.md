# ARIA Skills

Open-source skills for AI assistants built on [OpenClaw](https://github.com/openclaw/openclaw).

## Skills

### 🧠 memory-skill
Teaches your AI assistant how to use RAG memory effectively:
- Recall past conversations by date ("what did we discuss yesterday?")
- Query expansion for better semantic search
- Layered memory: hot (7 days) → daily summaries → monthly archive
- Honest responses when memory is missing

## Usage

Drop skill files into your assistant's system prompt or OpenClaw skills directory.
